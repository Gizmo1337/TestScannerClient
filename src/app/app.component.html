How to use me now


Dev. Build Info v0.09 Alpha

    nfo:

    An application that allows you to easily send barcodes to the ECPO database for test purposes without using a scanner and login to the ECPO Portal.

  1. Type in your full ScannerID (example: "000000000Pxxxxxx")
  2. Select the Gfk Server to work with
  3. Add in the table your Data
  4. Click on Send BarCodes
